4. C++ : The language that brought Object-Oriented Programming to the world

The First Inventor of C++

C++ is one of the most widely used programming languages, created as an extension of C. It was developed in 1979 by Bjarne Stroustrup.

1. Bjarne Stroustrup is the creator of C++.
2. Born on 30 December 1950 in Aarhus, Denmark.
3. Developed C++ at Bell Labs.
4. Initially called “C with Classes”.
5. Introduced Object-Oriented Programming to C.
6. First official version released in 1985.
7. Used in operating systems, games, and compilers.
8. Inspired modern languages like Java and C#.
9. Received the Charles Stark Draper Prize in 2018.
10. Celebrated as a legend of programming.

James Gosling – Influenced by C++

James Gosling

1. Creator of the Java programming language.
2. Inspired heavily by C++ concepts.
3. Added automatic memory management to improve safety.
4. Worked at Sun Microsystems.
5. His work revolutionized cross-platform software.
6. Designed Java to be simpler than C++.
7. Known as the “Father of Java”.
8. Java later became one of the most popular languages.
9. His work is seen as a continuation of Stroustrup’s legacy.
10. Honored as a pioneer in computing.

Herb Sutter – Modern Advocate of C++

Herb Sutter

1. Famous C++ expert and standards committee member.
2. Helped shape modern versions of C++.
3. Author of books like Exceptional C++.
4. Promotes safe and efficient coding practices.
5. Key figure in C++11, C++14, C++17, and C++20.
6. Known for simplifying complex concepts.
7. Advocates modern C++ in real-world programming.
8. Popular international speaker on C++.
9. Helps push C++ towards the future.
10. Respected leader in the C++ community.
